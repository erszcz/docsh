    docsh v0.7.1 rebar3_docsh

    Store modules' documentation in Docs chunks according to EEP-48.

    Add the following code to your project's rebar.config:

      {plugins,
   [
    {rebar3_docsh, "0.7.1", {pkg, docsh}}
   ]}.
 
  {provider_hooks,
   [
    {post, [{compile, {docsh, compile}}]}
   ]}.
